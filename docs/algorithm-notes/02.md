    # 画X

    绘制一个$X$(用*号表示线)，其中长、宽、对角线的长度(即可容纳的*号个数)均为同一个奇数$n$。

    **输入描述**
    一个正奇数$n$($3\le n\le 99$)。

    **输出描述**
    输出一个$X$。注意行末不要有多余的空格。

    **输入样本**：
    ```
    3
    ```

    **样本输出**：
    ```
    * *
     *
    * *
    ```

#include <bits/stdc++.h>
using namespace std;
const int N = 110;
char c[N][N];
int n;

int main()
{
    scanf("%d", &n);
    memset(c, 0, sizeof c);
    for(int i = 0; i < n; i ++)
    {
        int numSpace = (i < n / 2) ? (n - i) : (i + 1);
        for(int j = 0; j < numSpace; j ++)
        {
            c[i][j] = ' ';
        }
    }
    for(int i = 0; i < n; i ++)
    {
        c[i][i] = c[i][n - i - 1] = '*';
    }
    for(int i = 0; i < n; i ++)
    {
        puts(c[i]);
    }

    return 0;
}

### [日期加法](https://sunnywhy.com/sfbj/3/4/69)

::: collapse
- 点击展开题目
    # 日期加法

    给定一个日期$DAY$和一个正整数$n$，求日期$DAY$加上$n$天后的日期。

    **输入描述**
    第一行为给定的日期$DAY$(格式为YYYY - MM - DD，范围为1900 - 01 - 01$\le DAY\le $2199 - 12 - 31)，数据保证一定合法;

    第二行为需要增加的天数$n$($1\le n\le 10000$)。

    **输出描述**
    以YYYY - MM - DD的格式输出增加了$n$天后的日期。

    **输入样本**：
    ```
    2021-05-01
    30
    ```

    **样本输出**：
    ```
    2021-05-31
    ```

    > 代码长度限制[待补充] KB | 时间限制[待补充] ms | 内存限制[待补充] MB | 栈限制[待补充] KB
:::

**思路**
通过循环逐天增加日期，在循环中根据当前年份判断是否为闰年以确定2月的天数，当日期$dd$超过当前月份的天数时，将日期重置为1，并更新月份$mm$，若月份超过12，则更新年份$yyyy$ 。

::: code-tabs
@tab C++
```C++
#include <bits/stdc++.h>
using namespace std;
int months[13] = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30,31,30,31};
int yyyy, mm, dd;
bool isleap(int year)
{
    return ((year % 4 == 0) && (year % 100 != 0)) || (year % 400 == 0);
}

int n;

int main()
{
    scanf("%d-%d-%d", &yyyy, &mm, &dd);
    scanf("%d", &n);
    while(n)
    {
        if(isleap(yyyy)) months[2] = 29;
        else months[2] = 28;
        if(dd < months[mm])
        {
            dd ++;
        }
        else
        {
            dd = 1;
            if(mm < 12)
            {
                mm ++;
            }
            else
            {
                mm = 1;
                yyyy ++;
            }
        }
        n --;
    }
    printf("%04d-%02d-%02d", yyyy, mm, dd);
    return 0;
}
```

### [日期减法](https://sunnywhy.com/sfbj/3/4/70)

::: collapse

    # 日期减法

    给定一个日期$DAY$和一个正整数$n$，求日期$DAY$减去$n$天后的日期。

    **输入描述**
    第一行为给定的日期$DAY$(格式为YYYY - MM - DD，范围为1900 - 01 - 01$\le DAY\le $2199 - 12 - 31)，数据保证一定合法;

    第二行为需要减去的天数$n$($1\le n\le 10000$)。

    **输出描述**
    以YYYY - MM - DD的格式输出减少了$n$天后的日期。

    **输入样本**：
    ```
    2021-05-31
    30
    ```

    **样本输出**：
    ```
    2021-05-01
    ```

    > 代码长度限制[待补充] KB | 时间限制[待补充] ms | 内存限制[待补充] MB | 栈限制[待补充] KB
:::

**思路**
通过循环逐天减少日期，在循环中先根据当前年份判断是否为闰年，以确定2月的天数。若当前日期$dd$不为1，则直接减1；若$dd$为1，则将日期设为上一个月的最后一天，并更新月份$mm$，若月份为1，则更新年份$yyyy$ ，月份设为12 。

::: code-tabs
@tab C++
```C++
#include <bits/stdc++.h>
using namespace std;
int months[13] = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30,31,30,31};
int yyyy, mm, dd;
bool isleap(int year)
{
    return ((year % 4 == 0) && (year % 100 != 0)) || (year % 400 == 0);
}

int n;

int main()
{
    scanf("%d-%d-%d", &yyyy, &mm, &dd);
    scanf("%d", &n);
    while(n)
    {
        if(isleap(yyyy)) months[2] = 29;
        else months[2] = 28;
        if(dd != 1)
        {
            dd --;
        }
        else
        {
            if(mm != 1)
            {
                mm --;
            }
            else
            {
                mm = 12;
                yyyy --;
            }
            dd = months[mm];
        }
        n --;
    }
    printf("%04d-%02d-%02d", yyyy, mm, dd);
    return 0;
}
```
